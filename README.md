# Chalices of Fate
## Coded using Wing

# Gameplay
- Player is seeking treasure
- In order to gain the treasure, the player must choose one of three chalices to drink
- Player is either poisoned or recieves the treasure

# Code 
- uses time to display text and create a slow reveal of their fate
- takes player input and stores it in order to determine if they picked the good chalice
- uses random in order for the code to decide which of the three chalics is the good one
